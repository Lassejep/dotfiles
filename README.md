# dotfiles
Config files for my arch hyprland setup

- [zsh](https://www.zsh.org/)
- [tmux]
- [neovim](https://neovim.io/)
- [kitty](https://sw.kovidgoyal.net/kitty/)
- [hyprland]
- [dunst]
- [rofi-wayland]
- [waybar]
- [wlogout]
- [swappy]
- [grimblast]
- [zoxide]
- [pyenv]
- [yarn]
- [fzf]
- [ripgrep]
- [nerd-fonts-fira-code]
- [nerd-fonts-symbols]
- [lf]
- [tpm]
- [headsetcontrol]
- [bibata-cursor-theme]
- [yt-dlp]
- [mpv]
- [unzip]

## Installation
```bash
git clone https://github.com/lassejep/dotfiles.git
cd dotfiles
stow --dotfiles -t ~/ .
```
